# 项目结构

按照官方文档的内容，创建 `main.js` 和 `index.js` 并把相应内容复制进去，完成之后在工程的根目录下运行 `npm start`，
就可以看到一个简单的 `electron demo`，显示当前 `electron` 使用的 `chrome` `electron` 和 `nodejs` 版本

## 记事本的项目结构

- 项目入口脚本 `main.js` ，也是主进程脚本
- 主页面 `index.html` 
- 与主页面对应的渲染进程 `index.js`
- 可能需要的其他资源文件 `res` 文件夹
- 项目配置文件 `package.json`
